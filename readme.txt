Enables <a href="http://www.nickstakenburg.com/projects/lightview/" title="LightView">LightView</a> for images in posts and pages, including the WP 2.5 gallery.

== Description ==

This plugin will replace the WP 2.5+ gallery shortcode with code that utilizes Nick Stakenburg's LightView. A clean, simple JavaScript image and gallery view.

== Installation ==

1. Upload the `z-lightview` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Links to images and uses of the gallery shortcode will be updated to use LightView.
