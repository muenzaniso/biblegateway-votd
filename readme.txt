== Description ==

Adds <a href="http://www.biblegateway.com/">BibleGateway.com</a>'s verse of the day as a sidebar widget.  It will also allow you to include the verse of the day in a page or post using a shortcode. Different Bible translations can be selected from the widget's page.

== Installation ==

1. Upload the `biblegateway-votd.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
